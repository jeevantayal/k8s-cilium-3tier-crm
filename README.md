# k8s-cilium-3tier-crm
Kubernetes 3-tier CRM application using Cilium CNI with network policies


## ğŸ¯ Overview

This project implements a complete 3-tier CRM application on Kubernetes with:

- **Web Tier**: Nginx reverse proxy (3 replicas)
- **App Tier**: Python Flask REST API (3 replicas)
- **DB Tier**: PostgreSQL database (3 replicas, StatefulSet)
- **Security**: CiliumNetworkPolicy with zero-trust model
- **Load Balancing**: LoadBalancer service for external access
